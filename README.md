# ScheduleApp

Расписание занятий колледжа в телефоне — без сайтов, без вечных «куда делся файл в Яндекс.Диске»,
без сохранения .doc в загрузках. Открыл приложение — увидел расписание на сегодня.

---

## Что умеет приложение

**📚 Расписание для ученика**
Выбираешь день → выбираешь свою группу → видишь список пар: время, предмет, преподаватель,
кабинет. Текущая пара подсвечивается прямо во время занятия, а следующая — заранее, чтобы
успеть добежать до кабинета.

**🧑‍🏫 Расписание для преподавателя**
Тот же день, но наоборот: выбираешь себя из списка преподавателей — и видишь все свои пары
за день, с указанием, к какой группе идти и в какой кабинет.

**🔔 Звонки**
Расписание звонков — начало и конец каждой пары, отдельно.

**🎨 Темы оформления**
Тёмная, светлая и полностью чёрная (AMOLED) — под настроение и экран телефона.

**⚙️ Немного удобств**
- Приложение запоминает группу, чтобы не выбирать её каждый раз заново
- Можно вручную обновить список файлов, если расписание на диске обновилось только что
- Плавные анимации при переключении вкладок и открытии карточек

## Установка

1. Открой вкладку **[Releases](../../releases)** этого репозитория
2. Скачай файл `ScheduleApp-*.apk` из последнего релиза
3. Открой скачанный файл на телефоне

Если Android спросит про «неизвестные источники» — это нормально, приложение не из Google Play.
Разреши установку и продолжай.

> Требуется Android 8.0 и новее.

---

## Как пользоваться

1. При первом запуске открой ⚙️ **Настройки** и укажи ссылку на папку с расписанием на
   Яндекс.Диске (её даёт куратор/деканат) — без этого список дней будет пустым.
2. На главном экране выбери день из списка файлов.
3. Переключателем **«Ученики / Преподаватели»** над списком выбери, кто ты.
4. Выбери свою группу (или себя — если ты преподаватель) из появившегося списка.
5. Готово — расписание на экране. Кнопка ✎ в шапке позволяет сменить группу/преподавателя,
   не выходя с экрана расписания.

Выбор группы или преподавателя не сохраняется по умолчанию для расписания преподавателя —
только для группы (в Настройках это можно включить/выключить).
